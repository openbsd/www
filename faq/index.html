Frequently Asked Questions

This FAQ is supplemental documentation to the man pages, available both in the installed system and online. The FAQ covers the active release of OpenBSD, currently v5.8. There are likely features and changes to features in the development version (-current) of OpenBSD that are not covered in the FAQ.
