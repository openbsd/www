Documentation and Frequently Asked Questions

Commonly Encountered Issues          Recent updates

This FAQ is supplemental documentation to the man pages, available both in the installed system and online. The FAQ covers the active release of OpenBSD, currently v3.3. Note that the development version (-current) of OpenBSD is not covered by this FAQ.

The FAQ is available in PDF and plain text form in the pub/OpenBSD/doc directory from the FTP mirrors.
