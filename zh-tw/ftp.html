檔案目錄架構:

每個伺服器的檔案架構大體上如下:

  • .../OpenBSD/3.4/
    擺放目前最新的正式版.
    • alpha/
    • hp300/
    • hppa/
    • i386/
    • mac68k/
    • macppc/
    • mvme68k/
    • mvme88k/
    • sparc/
    • sparc64/
    • vax/
    • src.tar.gz
       使用者程式原始碼，請放到 /usr/src. (不包含核心原始碼)
    • sys.tar.gz
       核心原始碼，請放到 /usr/src/sys.
    • ports.tar.gz
       ports 的原始碼，請放到 /usr/ports.
    • XF4.tar.gz
       XFree86 (X11) 原始碼.

  • .../OpenBSD/snapshots/
    對於主要的支援平台，我們傾向大約每個月會製作小修改的釋出版，這些釋出版 的穩定性及品質是不穩定的. 這是我們存放測試釋出版的地方.

  • .../OpenBSD/src/
    這是每天從我們的原始碼透過 cvs 取出，為了可以讓那些想存取個別檔案的 人使用. 一般來說我們建議使用 CTM 或 AnonCVS 而不要直接映射這個位置. 因此, 請所有個 ftp 映射站台不要複製這個位置.

  • .../OpenBSD/doc/
    OpenBSD 常見問答, 包括 PDF 及純文字版本, 其中涵蓋了其他 OpenBSD 相關的文件資料.

  • .../OpenBSD/tools/
    一些小工具.
  • .../OpenBSD/3.3/
    檔案架構類似於以上所述的 3.4 正式版. 為了延用舊版本架構.
  • .../OpenBSD/3.2/
    檔案架構類似於以上所述的 3.4 正式版. 為了延用舊版本架構.
  • .../OpenBSD/3.1/
    檔案架構類似於以上所述的 3.4 正式版. 為了延用舊版本架構.
  • .../OpenBSD/3.0/
    檔案架構類似於以上所述的 3.4 正式版. 為了延用舊版本架構.
  • .../OpenBSD/2.9/
    檔案架構類似於以上所述的 3.4 正式版. 為了延用舊版本架構.

想要建立新的 FTP/ANONCVS 映射站台的一定會對以下的磁碟空間使用估計有興趣, 這是由 ftp://ftp.openbsd.org:/pub/OpenBSD/所維護的.
目前要列在本頁的映射站台至少需要提供兩個正式釋出版(目前是 3.2 及 3.3), 需包含所有的平台及 OpenSSH/, patches/ 及 tools/ 等目錄.
根據您的磁碟空間, 您可以容納更多的內容在您的映射裡, 包括搶鮮版, 更舊的釋出版, 目前的原始碼等等.
目前主要 ftp 站的磁碟空間利用如下: 


                      current  could be
pub/OpenBSD/2.9        3.0GB     3.0GB   (OpenBSD 2.9 release)
pub/OpenBSD/3.0        4.5GB     4.5GB   (OpenBSD 3.0 release)
pub/OpenBSD/3.1        6.9GB     6.9GB   (OpenBSD 3.1 release)
pub/OpenBSD/3.2       11.8GB    11.8GB   (OpenBSD 3.2 release)
pub/OpenBSD/3.3       14.0GB    14.0GB   (OpenBSD 3.3 release)
pub/OpenBSD/3.4       11.0GB    14.0GB   (OpenBSD 3.4 release)
pub/OpenBSD/snapshots  8.5GB    更大!  (目前的搶鮮版, 通常佔用空間會大於
					最新的發行版本)
pub/OpenBSD/src         .6GB      .7GB   (可用 ftp 取得的原始碼, 每晚從 
			  		 anoncvs 取出.你應該避免使用 ftp
                                         映射站台來取得. 最好是從離您最近的 
                                         anoncvs 伺服器來進行更新.)
因此, 一個完整的映射需要大約 50GB 的磁碟空間, 但是如果只是最新的兩個正式版, 只需要 25GB 即可. 注意, 以下主要 ftp 站所提供的目錄, 其他映射站台可能不會提供: 

pub/OpenBSD/distfiles  4.4GB   更大!  (distfiles 是由 ports 子系統所使用的
                                         makefiles. 您應該不用
					 複製這些除非您有的其他理由.
					 這些 ports makefiles 預設只會察看
					 ftp.openbsd.org 和 ftp.usa.openbsd.org)

如果您新增了一個新的 ftp, http, AFS 或 rsync 映射站, 請聯絡 www@openbsd.org.
