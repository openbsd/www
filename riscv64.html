OpenBSD riscv64

OpenBSD/riscv64 is a port of OpenBSD to various 64-bit RISC-V based systems.

Current status

The current target platform is the HiFive Unmatched board.

When booting the HiFive Unmatched board, write an install/miniroot image to a USB flash drive, and boot with that drive plugged in. Make sure you also have the microSD card plugged in that shipped with the HiFive Unmatched board. The board should auto-boot the OpenBSD bootloader, which will load the bsd.rd RAMDISK install kernel.

Supported hardware

OpenBSD/riscv64 runs on the following hardware:

  • Allwinner D1
    • Mango Pi MQ-Pro
    • Nezha
  • SiFive
    • HiFive Unmatched
  • StarFive
    • StarFive VisionFive and VisionFive 2
    • Note: Some firmware configuration tweaks are necessary to boot OpenBSD
  • Microsemi
    • PolarFire SoC Icicle Kit
    • Note:PolarFire SoC Icicle Kit support is incomplete

Ethernet

  • cad(4) integrated Cadence Gigabit Ethernet controller
  • dwqe(4) Synopsys DesignWare Ethernet Quality-of-Service Controller
  • dwxe(4) integrated Allwinner EMAC 10/100/Gigabit Ethernet on Allwinner D1

Storage

  • dwmmc(4) on StarFive VisionFive/VisionFive 2
  • nvme(4) on HiFive Unmatched and VisionFive 2

USB

  • xhci(4) on HiFive Unmatched and StarFive VisionFive/VisionFive 2
