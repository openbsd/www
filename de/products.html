Produkte für OpenBSD

OpenBSD ermutigt Firmen und unabhängige Entwickler dazu, Produkte zu entwickeln, die unter OpenBSD nutzbar sind, oder auf OpenBSD basieren.

Kommerzielle Unterstützung für OpenBSD ist von Anbietern verfügbar, die auf unserer Supportseite aufgelistet sind.

  • Produkte, die auf OpenBSD basieren
  • Hardware-/Softwareprodukte für OpenBSD

Produkte, die auf OpenBSD basieren

Die folgenden Produkte basieren auf OpenBSD:

  • nCircle Network Security
    nCircle Network Security (vormals Hiverworld) bietet umfassende Netzwerksicherheit und geht dabei über Intrusion Detection hinaus, hin zu Intelligenter Intrusion Prevention. Ihr Patentantrag ist noch in der Schwebe, doch ihre IP360-Lösung liefert dauerhafte nicht invasive Verwundbarkeitsuntersuchungen kombiniert mit Traffic Monitoring und Intrusion Analysis mit der vollen Geschwindigkeit des Netzwerks. nCircle hat sich für seinen Scanning Sensor, Intrusion Sensor und seine Console Engine für OpenBSD als Betriebssystem entschieden, die alle auf einem sicheren Netzwerkgerät laufen.

  • RTMX
    RTMX verkauft eine Version von OpenBSD, die eine vollständigen Zusatz an POSIX-,real-time'-Funktionalitäten hat. Sie haben diese uns diese Änderungen großzügig zur Verfügung gestellt, und die Änderungen werden bald in OpenBSD integriert werden.

  • syscall Network Solutions
    Syscall Network Solutions produziert die Sicherheitsanwendung syswall basierend auf OpenBSD. Die syswall-Sicherheitsanwendung beinahltet eine ,stateful packet-filtering' Firewall, ein Intrusion-Detection-System und VPN-Fähigkeiten sogar mit dynamischen IP-Addressen. Sie wurde speziell designed, um Managed Security Services und Managed Security Monitoring von unseren IT-Sicherheitsexperten zu bieten. Die syswall kann also als DSL- oder Kabel-Router benutzt werden, da sie PPPoE und DHCP unterstützt.

  • GeNUA
    GeNUA mbH, eine Firma mit Sitz in München, Deutschland, die ausgeklügelte IT-Sicherheitslösungen anbietet, die auf OpenBSD basieren: Das Two-tier-GeNUGate-System ist die einzige Firewall, die vom Bundesamt für Sicherheit in der Informationstechnik (BSI) auf E3 / hohes Level unter dem internationalen ITSEC-Standard zertifiziert wurde. Das Kryptosystem GeNUBox wurde für Fernwartung von Maschinen über das Internet entwickelt. Neben anderen ist eine wichtige Funktion von diesem GeNUA-Produkt die Erstellung von SSH-verschlüsselten Tunneln für Datenübertragung über das Internet und das Anbieten einer integrierten Applikationsplattform.

Hardware-/Software-Produkte für OpenBSD

Von den folgenden Produkten ist bekannt, dass sie mit OpenBSD zusammen funktionieren:

  • PowerCrypt von GTGI
    Global Technologies Group, Inc. produziert PowerCrypt, eine intelligente bus-masterfähige PCI-Karte, die auf dem Hifn-7751-Verschlüsselungsprozessor basiert. Kernel-Level-Unterstützung wurde von OpenBSD entwickelt, Gerätetreiber und IPsec-Code wurde von von Mitgliedern der OpenBSD-Gemeinschaft geschrieben.

  • RedHat's Stronghold Secure Web Server
    RedHat bietet einen 128-Bit starken Kryptographie-HTTPS-Daemon basierend auf dem weit verbreiteten Apache Web Server auf OpenBSD mit ihrem preisgekrönten Stronghold 3 Secure Web Server.

  • Open Sound System von 4Front (nur i386)
    4Front Technologies bietet eine wesentlich verbesserte kommerzielle Version der OpenBSD/i386-Audiotreiber. Diese Ersatztreiber sind kompatibel mit denen, die mit OpenBSD/i386 ausgeliefert werden, aber sie unterstützen zusätzliche Karten und auch deren komplette Funktionalitäten. Dieses Paket ist für OpenBSD/i386 3.x erhältlich.

  • Core Systems
    Core Systems verkauft InSite, ein einfach zu benutzendes serverseitiges Webstatistik-Werkzeug für OpenBSD/i386. InSite ist Produkten wie WebTrends ähnlich, ist aber auch durch ein Webinterface zu konfigurieren, um Berichte während des Betriebs zu erstellen, und es verbraucht dabei kaum Rechenzeit. (Auf Nachfrage ist Core in der Lage, InSite auch für andere Plattformen als i386 bereitzustellen.)

  • Software2Go Motif (nur i386 und SPARC)
    Software2Go, LLC hat Motif-2.1.20-Entwicklungs- und Runtime-Toolkits für OpenBSD.
    Die Entwicklungsdistribution ist für Softwareentwickler, die grafische Oberflächen mit dem look-and-feel von Motif entwickeln. Die Runtime-Distribution ist notwendig, um nur lauffähige Applikationen zu entwickeln, also ohne Quellcode.
    Kontaktiere: Software2Go, LLC Online Bestellung, 817-431-8775 (phone/fax)

  • My Restaurant
    My Restaurant ist ein vollständig integriertes Hardware-/Software-Produkt, das Restaurants ermöglicht, jeden Teilaspekt ihres Geschäftes zu verwalten (von der Kasse bis zum Versorgungskettenmanagement und alles dazwischen). OpenBSD ermöglicht wbp systems, ein Produkt zu entwickeln, das unglaublich stabil ist, was an OpenBSDs exzellentem Design liegt.

  • Legato CRM
    wbp systems' Legato CRM ermöglicht es dem drahtlosen Geschäftsführer, Abrechnungen, Inventarisierung, Mitarbeiter- und Kundenverwaltung von ein und der selben Stelle aus durchzuführen. Als ein integriertes Hardware-/Software-Produkt baut Legato auf OpenBSDs solider Leistung und Einsatzflexibilität.
