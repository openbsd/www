OpenBSD/sun3

Der OpenBSD/sun3 port läuft zur Zeit auf allen 68020-basierten Sun3 Computern, mit Ausnahme der 3/E und der 3/50 Varianten.

Der OpenBSD/sun3 port wurde nach der Version 2.9 nicht weiterentwickelt.

Zur Zeit arbeitet niemand daran, diesen port aktiv wieder zum Leben zu erwecken. Der Source Code ist nach wie vor über CVS Attic zu bekommen, genauso wie welcher in NetBSD zu finden ist. Wenn du den sun3 port wieder zum Leben erwecken willst, bringe ihn zum Laufen und wende dich danach an Miodrag Vallat.

Geschichte:

Der sun3 port von OpenBSD wurde vom NetBSD/sun3 port abgeleitet, als die sun3 und sun3x ports noch verschiedene ports waren. Eine Zeitlang hat niemand echtes Interesse an OpenBSD/sun3 gezeigt, obwohl der Code grösstenteils mit dem Code aus NetBSD synchron gehalten wurde.

Irgendwann begann Kenneth Stailey am port zu arbeiten, aber da er in den USA lebte, konnte er keine der Version exportieren, die er erzeugt hatte, und verlor so bald das Interesse an diesem port.

Zu Beginn des Jahres 2000 versuchte Miodrag Vallat den port wieder in Form zu bringen, brachte den Code wieder zum Compilieren und erzeugte vollkommen neue Installations-Medien. Der port wurde sogar mit komplettem X Window System Client und Utilities ausgeliefert, obwohl es keinen funktionierenden X Server gab.

Der port lebte dann noch ein paar Version lang, aber Hardwareprobleme und der Mangel an Zeit führten dazu, dass der Anschluss an NetBSD verpasst wurde und der sun3 code war noch nicht bereit für das Umstellen vom alten Mach virtuellen Speichersystem zu uvm, und die Veröffentlichung von Version 3.0 wurde verpasst, in der Hoffnung, dass der Code zur Veröffentlichung von 3.1 fertig wäre. Leider war das nicht der Fall, und so wurde beschlossen, die Arbeit daran einzustellen und den Code aus dem Source Tree zu entfernen.

Unterstützte Hardware:

Unterstützte Modelle

  • 3/60, 3/60LE
  • 3/75, 3/140, 3/150, 3/160 und 3/180
  • 3/110
  • 3/260 und 3/280
OpenBSD läuft nicht auf den 3/160 Maschinen mit nur 2MB Speicher, und kaum auf denen mit 4MB, egal welches Modell.

Unterstützte Peripheriegeräte

  • On-board serielle Ports
  • Sun type 3 Tastatur und Maus
  • On-board oder VME Grafik(bwtwo, cgtwo, cgfour)
  • On-board oder VME Ethernet (intel, lance)
  • On-board oder VME sun-3 SCSI Controller, momentan mit einigen Einschränkungen
