OpenIKED Releases

The OpenIKED project tracks OpenBSD development. Stable releases are derived from the most recent OpenBSD release, plus security updates as necessary.

Supported Platforms

OpenIKED Portable is known to work on the following platforms:
  • Linux
  • FreeBSD
  • NetBSD
  • macOS
