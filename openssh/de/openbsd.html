OpenSSH für OpenBSD

Da OpenSSH von der OpenBSD-Gruppe entwickelt wird, ist OpenSSH Teil der Basis des Betriebssystems, beginnend ab OpenBSD Version 2.6 und aufwärts. Es gibt also nichts, was man dem System hinzufügen oder daran ändern müsste. Nichtsdestotrotz möchtest du vielleicht aufgrund neuer Funktionen ein Upgrade machen.

Du kannst die aktuellste OpenSSH-Version von diesen FTP-,mirrors' herunterladen.

Du kanst die folgenden Versionen von OpenSSH auf deinen OpenBSD-Maschinen installieren:

  • OpenSSH 3.9: 17. August 2004

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-3.9.tgz

    Wenn du OpenSSH 3.9 auf OpenBSD 3.5 installierst, brauchst du den folgenden Patch:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_3.9.patch.

    Wenn du OpenSSH 3.9 auf einer äteren Version installierst, brauchst du den folgenden Patch:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd34_3.9.patch.

    
# cd /usr/src/usr.bin
# tar xvfz .../openssh-3.9.tgz
# cd ssh
# patch -p0 < /pfad/zu/openbsd3x_3.9.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    und starte sshd neu. Um sshd(8)s neue ,privilege separation'-Funktionalität auf einigen älteren Versionen zu unterstützen, wurden dem System ein neuer User und eine neue Gruppe namens sshd hinzugefügt. Füge die folgenden Einträge mittels vipw(8) hinzu: 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    Füge das folgende in /etc/group ein: 

    sshd:*:27:
_sshagnt:*:34:
