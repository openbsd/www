Java

Die folgenden "freien" Clients werden für die Zusammenarbeit mit OpenSSH von Java Clients aus empfohlen:

  • MindTerm 1 ist eine nur-SSH1 Implementation. GPL Lizenz.

    "MindTerm ist ein vollständig freies(*) SSH (Protokoll 1) Client Programm , dass in 100% purem Java geschrieben wurde (nicht-zertifiziert). Sie wird mit Source Code ausgeliefert, da wir glauben dass sicherheitsrelevanter Software ansonsten nicht getraut werden kann. Mindterm kann als stand-alone Programm oder als Applet in einer Webseite benutzt werden. Es kann mit oder ohne GUI (Graphical User Interface) benutzt werden (stand-alone). Es hat zwei besondere Features, die andere Clients nicht haben: eingebauter scp-File Transfer und einen speziellen ftp-tunnel, der mit "ordinären" ftpd's hinter sshd arbeiten kann."

    HINWEIS: neuere Versionen sind nicht frei.

  • Java-SSH ist eine nur-SSH1 Implementation von Cédric Gourio. Unterliegt einer verwirrenden Mixtur von nicht-kommerziellen und GPL Lizenzen.

    "Dieses Applet implementiert das Protokol 1.5. Die Terminal Emulation ist fast vollständig VT100. Es bietet sichere Kommunikationen über unsichere Netzwerke. Alle Kommunikationen werden automatisch und transparent verschlüsselt. Verschlüsselung wird auch zur Sicherstellung der Integrität benutzt."

  • SSH Plugin ist eine nur-SSH1 Implementation von Matthias L. Jugel und Marcus Meißner. GPL Lizenz.

    "Diese Implementation von SSH bietet IDEA und RSA/PKCS#1 Verschlüsselung und wurde in seiner Originalform 1998 von Cédric Gourio geschrieben. Er nutzte einen Teil des alten Java Telnet Applet um seinen eigenen Java SSH Client zu schreiben und daher war der Code unserem alten Applet sehr ähnlich."
