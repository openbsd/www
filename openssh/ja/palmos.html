Palm OS

以下の「フリー」クライアントが Palm OS クライアントで OpenSSH と相互運用できるものとしておすすめです:

  • Top Gun ssh (pilotSSLeay 使用)

    「Top Gun ssh は TCP/IP スタックをもっていて、Palm OS を走らせている palmtop 上で動く ssh (バージョン 1) のクライアントです。 ここには Palm Pilot Pro, Palm III, Palm V および Workpad の これらに対応するモデル、そして Visor が含まれます。Palm VII は含まれません。 この機種では TCP/IP を扱う方法がないようです」

  • pssh

    「pssh はフリーで、オープンソースの Palm OS 用 SSH 2 クライアントです。 これは Palm OS 5 と ARM プロセッサ、および 320x200 以上の解像度の画面をもつ ハンドヘルド機種上で動きます」

    これは SSH プロトコル バージョン 2 をサポートしています。 ソースコードも利用可能です。OpenSSH の一部を利用しています。

  • TuSSH

    「TuSSH は Palm OS 用の SSH クライアントです。今ではこれは Palm OS 4 以上の 機種で動くはずです。OS 3.5 の機種でさえ動くかもしれませんが、この機種での テストやサポートは予定されていません」

    SSH プロトコル バージョン 1 と、バージョン 2 の一部をサポートしています。 使用するのは無料ですが、ソースコードは利用可能ではありません。
