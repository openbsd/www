OpenBSD 専用

OpenSSH は OpenBSD グループによって開発されているため、これは OpenBSD リリース 2.6 以降の基本オペレーティングシステムに 含まれています。ですから、お使いのシステムに追加したりインストールしたり する必要はありません。けれども新しい機能やその他のパッチのために アップグレードしたいこともあるでしょう。

OpenSSH の最新リリースは ftp ミラーサイト から ダウンロードできます。

以下の OpenSSH のバージョンを、お使いの OpenBSD マシン上で 動かすことができます:

  • OpenSSH 5.0: 2008年 4月3日

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-5.0.tar.gz 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-5.0.tgz
# cd ssh
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
    この後 sshd を再起動してください。

    OpenSSH 5.0 を、OpenBSD 4.3, 4.2 あるいは 4.1 上にインストールする場合は、 以下のパッチが必要です:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd43_5.0.patch 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-5.0.tgz
# cd ssh
# patch -p0 < /path/to/openbsd43_5.0.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    この後 sshd を再起動してください。

  • OpenSSH 4.0: 2005年 3月9日

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-4.0.tgz

    OpenSSH 4.0 を OpenBSD 3.5 にインストールする場合は、 以下のパッチが必要です:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_4.0.patch 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-4.0.tgz
# cd ssh
# patch -p0 < .../openbsd3x_4.0.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    この後 sshd を再起動してください。 OpenBSD のいくつかの古いリリースでは、 sshd(8) の root 権限分離機能をサポートするために、新しいユーザとグループ sshd が システムに追加されています。 vipw(8) を使って、以下のユーザエントリを追加してください: 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    以下を /etc/group に追加してください: 

    sshd:*:27:
_sshagnt:*:34:
