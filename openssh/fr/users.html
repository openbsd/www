Systèmes utilisant OpenSSH

Les systèmes d'exploitation et produits suivants sont connus pour avoir intégré OpenSSH dans le système de base.

Cette liste est dans l'ordre chronologique. Le plus tôt un système a intégré OpenSSH le plus tôt il est listé.

  • OpenBSD
  • Debian Linux
  • FreeBSD
  • Suse Linux
  • Redhat Linux
  • Mandrake Linux
  • BSDi BSD/OS
  • NetBSD
  • Computone
  • Conectiva Linux
  • Slackware Linux
  • Caldera OpenLinux
  • Stallion
  • Rock Linux
  • Cygwin
  • e-smith server and gateway
  • Engarde Linux
  • Mac OS X Version 10.1 et ultérieur
  • HP Procurve Switch 4108GL and 2524/2512
  • IBM AIX
  • Gentoo Linux
  • Gwynux/Toadware Linux
  • Sun Solaris 9 et ultérieur (appelé SunSSH)
  • SmoothWall
  • IPCop
  • SGI Irix
  • ThinLinc
  • Nokia IPSO
  • Cisco CSS11500 series content services switches
  • Cisco SN 5400 series storage routers
  • TopLayer IDS balancers
  • Commutateur Port Série SSH NTI
  • Bluecoat (anciennement Cacheflow) Proxy SG
  • Novell NetWare
  • Digi CM Console Servers
  • Fedora Linux
  • Alcatel OmniSwitch

Si nous avons oublié un système qui intégre OpenSSH dans le système de base, merci de le porter à notre connaissance. (OpenSSH doit être inclus par défaut soit sur le CD soit dans le système et ne requiert pas une intégration de la part de l'utilisateur).
