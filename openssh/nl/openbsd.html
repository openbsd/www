OpenSSH voor OpenBSD

Aangezien OpenSSH ontwikkeld wordt door de OpenBSD groep is OpenSSH standaard bij het besturingssysteem inbegrepen sinds OpenBSD versie 2.6. Er hoeft dus niets geinstalleerd of toegevoegd te worden aan uw systeem. Maar u wilt wellicht upgraden voor nieuwe functionaliteiten of andere patches.

U kunt de huidige OpenSSH versie downloaden van deze ftp mirrors.

De volgende versies van OpenSSH zijn beschikbaar voor u om op uw OpenBSD machines te zetten:

  • OpenSSH 4.2: 26 mei 2005

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-4.2.tar.gz

    Als u OpenSSH 4.2 op OpenBSD 3.5 installeert, hebt u de volgende patch nodig:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_4.2.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-4.2.tar.gz
# cd ssh
# patch -p0 < /pad/naar/openbsd35_4.2.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
    en start sshd opnieuw op.

  • OpenSSH 4.0: 9 maart 2005

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-4.0.tgz

    Als u OpenSSH 4.0 op OpenBSD 3.5 installeert, hebt u de volgende patch nodig:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_4.0.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-4.0.tgz
# cd ssh
# patch -p0 < /pad/naar/openbsd3x_4.0.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    en start sshd opnieuw op. Om de privilege separation optie van sshd(8) op enkele oudere uitgaven van OpenBSD te ondersteunen zijn een nieuwe gebruiker en groep sshd genaamd toegevoegd aan het systeem. Voeg de volgende gebruiker toe met vipw(8): 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    Voeg het volgende toe aan /etc/group:

    sshd:*:27:
_sshagnt:*:34:
