OpenSSH для OpenBSD

С тех пор как OpenSSH разрабатывается группой OpenBSD, она входит в базовую поставку ОС, начиная с версии 2.6. Таким образом, не составляет никакого труда установить ее на вашу систему. В связи с периодическим добавлением новых функций или просто исправлений, Вы также легко можете ее обновить.

Вы можете загрузить актуальную версию OpenSSH с FTP-зеркал.

Эти версии OpenSSH могут быть установленны на OpenBSD машинах:

  • OpenSSH 5.9: 6 сентября 2011

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-5.9.tar.gz

    При установке OpenSSH 5.9 на OpenBSD 4.8 или 4.9 Вам понадобится один из этих патчей:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd48_5.9.patch для OpenBSD 4.8, или ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd49_5.9.patch для OpenBSD 4.9. 

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-5.9.tar.gz
# cd ssh
# patch -p0 < /path/to/openbsd49_5.9.patch  # или openbsd48_5.9.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
    и перезапускаем sshd.

  • OpenSSH 4.0: 17 августа 2004

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-4.0.tgz

    При установке OpenSSH 4.0 на OpenBSD 3.5 используй этот патч:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_4.0.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-4.0.tgz
# cd ssh
# patch -p0 < /pfad/zu/openbsd3x_4.0.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    и перезапускаем sshd. Для поддержки sshd(8) с функцией разделения полномочий в некоторых старых версиях OpenBSD, надо добавть нового пользователя и группу, названную sshd. Добавьте следующую запись пользователя с помощью vipw(8): 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    А также добавьте это в /etc/group:

    sshd:*:27:
_sshagnt:*:34:
