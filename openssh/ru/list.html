Списки рассылки

Объявления

Список рассылки (только для чтения) openssh-unix-announce@mindrot.org для Linux/Unix порта OpenSSH. Список рассылки предназначен для извещения о новых версиях программного обеспечения.

Чтобы подписаться, отправьте письмо на openssh-unix-announce-request@mindrot.org содержащее в теле "SUBSCRIBE". Вам необходимо будет подтвердить подписку, перед тем как ваш запрос обработают.

Чтобы отписаться, отправьте письмо на openssh-unix-announce-request@mindrot.org содержащее в теле "UNSUBSCRIBE". Вам также необходимо отослать подтверждение, перед тем как вас отпишут от списка рассылки.

Список разработчиков

Этот список рассылки openssh-unix-dev@mindrot.org предназначен для общения разработчиков, желающих усовершенствовать порт и/или сделать поддержку для другой платформы. Этот список является хорошим местом для доклада и обсуждения ошибок.

Список рассылки открыт для не подписанных пользователей. НЕ РАЗРЕШАЕТСЯ использовать HTML. Разрешается пересылать маленькие файлы (например diff файлы). Архив доступен на MARC.

Чтобы подписаться, отправьте письмо на openssh-unix-dev-request@mindrot.org содержащее в теле "SUBSCRIBE". Вам необходимо отослать подтверждение, перед тем как ваш запрос обработают.

Чтобы отписаться, отправьте письмо на openssh-unix-dev-request@mindrot.org содержащее в теле "UNSUBSCRIBE". Вам также необходимо отослать подтверждение, перед тем как вас отпишут от списка рассылки.

Основной список обсуждения SSH

Основной список рассылки SSH secureshell@securityfocus.com. Вы также можете использовать группу новостей Usenet comp.security.ssh.

Чтобы подписаться, отправьте пустое сообщение по адресу secureshell-subscribe@securityfocus.com.

Чтобы отписаться, отправьте пустое сообщение по адресу secureshell-unsubscribe@securityfocus.com.

Архив доступен на MARC.
