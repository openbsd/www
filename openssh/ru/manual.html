Документация

На Web доступны man страницы для нижеперечисленных команд. Эти страницы относятся к последней версии OpenSSH.

  • ssh - Основная программа клиент, наподобие rlogin/rsh.
  • sshd - Серверная программа, позволяет входить в систему.
  • ssh-agent - Агент аутентификации, может хранить ваши приватные ключи.
  • ssh-add - Утилита для добавления ключей к агенту.
  • sftp - Программа наподобие FTP, работает через протоколы SSH1 и SSH2.
  • scp - Программа копирования файлов, работает подобно rcp(1).
  • ssh-keygen - Утилита для создания ключей.
  • sftp-server - SFTP сервер (автоматически запускается из под sshd).
  • ssh-keyscan - Утилита для получения публичных ключей с нескольких машин.
  • ssh-keysign - Утилита для hostbased аутентификации.

Реализация протокола SSH2 в OpenSSH стандартизована рабочей группой IETF secsh и заключена в нескольких проект документах. Общая структура SSH2 протокола описана здесь. Протокол состоит из трёх уровней:

  • Транспортный уровень обеспечивает взаимодействие алгоритма и обмен ключами. Обмен ключами позволяет аутентифицировать сервер и создать криптографически защищённое соединение: обеспечивая целостность, конфиденциальность и дополнительное сжатие.
  • Уровень аутентификации пользователя использует установленное соединение и располагается на транспортном уровне. Этот уровень предоставляет несколько механизмов для аутентификации пользователя: здесь может использоваться традиционная парольная аутентификация, аутентификация основанная на публичном ключе и т.д.
  • Уровень соединения объединяет множество параллельных каналов, располагается на аутентификационном уровне и позволяет создавать login сессии, либо пересылать TCP соединения. Обеспечивает корректность передачи для этих каналов.
Дополнительные документы:
  • Интерактивная аутентификация описывает новые схемы аутентификации, такие как S/Key или TIS.
  • Протокол для передачи файлов SFTP описан в документе filexfer. В OpenSSH реализован клиент и сервер SFTP.
  • Формат файла для публичных ключей определён в документе publickeyfile. Вы можете преобразовать OpenSSH файл с публичными ключами в этот формат используя утилиту ssh-keygen.
