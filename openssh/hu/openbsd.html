OpenSSH OpenBSD-re

Mivel az OpenSSH-t az OpenBSD csapat fejleszti, az OpenSSH része az alap operációs rendszernek az OpenBSD 2.6-os kiadása óta. Így nem kell semmit telepítened vagy hozzáadnod a rendszeredhez. Azonban az újabb szolgáltatások és hibajavítások miatt szükség lehet a frissítésre.

A jelenlegi OpenSSH kiadás letölthetõ ezekrõl az ftp tükrökrõl.

A következõ OpenSSH verziók elérhetõk OpenBSD-re:

  • OpenSSH 4.0: 2005. március 9.

    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openssh-4.0.tgz

    Ha OpenSSH 4.0-et telepítesz OpenBSD 3.5-re, akkor a következõ foltra lesz szükséged:
    ftp://ftp.openbsd.org/pub/OpenBSD/OpenSSH/openbsd35_4.0.patch.

    # cd /usr/src/usr.bin
# tar xvfz .../openssh-4.0.tgz
# cd ssh
# patch -p0 < /elérési/út/openbsd3x_4.0.patch
# make obj
# make cleandir
# make depend
# make
# make install
# cp ssh_config sshd_config /etc/ssh
# mkdir /var/empty
    és indítsd újra az sshd-t.
    Az sshd(8) jogosultság elkülönítés (privilege separation) szolgáltatásának támogatásához néhány régebbi OpenBSD kiadásban, egy új sshd nevû felhasználó és csoport lett hozzáadva a rendszerhez. A felhasználó hozzáadása vipw(8) segítségével: 
    sshd:*:27:27::0:0:sshd privsep:/var/empty:/sbin/nologin

    Add a következõket az /etc/group fájlba:

    sshd:*:27:
_sshagnt:*:34:
