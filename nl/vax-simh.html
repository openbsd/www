OpenBSD/vax op SIMH

Het is erg gemakkelijk om OpenBSD/vax op de SIMH VAX simulator te installeren. Het enige wat u nodig heeft is een redelijk snelle machine, genoeg ruimte op uw harde schijf voor het 1.5GB grote disk image en het floppy37.fs bestand van uw CD-ROM set. Er van uitgaande dat u SIMH op een OpenBSD systeem gaat draaien, installeer dan het emulators/simh package via de ports. Zoek een goede locatie voor uw OpenBSD/vax SIMH omgeving en maak op deze locatie het bestand openbsd.simh aan met de volgende inhoud: 

set cpu 64m
at nvr openbsd.nvram

deposit rq qtime 100

set rq0 ra92
at rq0 openbsd.ra0.disk

set rq1 cdrom
at rq1 /dev/rcd0c

set rq2 cdrom
at rq2 floppy37.fs

at xq0 bge0

boot cpu
exit
Dit zal SIMH zo configureren dat het een VAX simuleert met een uitzonderlijk 64MB geheugen, een RA92 1.5GB schijf, een CD-ROM station voor CD1 van uw CD-ROM set, een ander CD-ROM station om vanaf te booten en een DELQA Qbus Ethernet controller. U zult bge0 moeten vervangen met de naam de netwerk interface van uw computer. Ook zult u misschien /dev/rcd0c moeten vervangen met de naam van het raw device wat overeenkomt met uw echte CD-ROM station. Zie de SIMH documentatie voor meer details.

Kopiëer CD1:3.7/vax/floppy37.fs naar de locatie van uw OpenBSD/vax SIMH omgeving. Unmount CD1, maar laat het nog wel in uw CD-ROM station zitten. Start nu SIMH. Type op het boot prompt boot dua2:, nu zal SIMH als het goed is het normale OpenBSD installatie proces starten:

Nadat u de installatie heeft voltooid, herstart dan SIMH en type nu: 
>>>boot dua0:
om het nieuwe disk image te booten. U heeft wat geduld nodig bij de eerste boot, want de SSH host keys zullen gegenereerd worden en dit kan even duren. Maar uiteindelijk zal het boot proces voltooid worden en u zult een vriendelijke login prompt gepresenteerd krijgen: 
OpenBSD/vax (noname.my.domain) (console)

login:
Als u klaar bent met het installeren van eventuele packages, dan zult u misschien de regels willen verwijderen, uit het OpenBSD.simh bestand die verwijzen naar uw echte CD-ROM station en floppy37.fs.
