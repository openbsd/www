  • 五种 体系结构的完整安装组件: i386, macppc, vax, sparc, sparc64.
  • 如下体系结构只能通过 FTP 下载: alpha, hppa, hp300, mvme68k, mac68k.
  • 在 i386, macppc, sparc, 和sparc64下,是可以从光盘引导的.
  • 一张漂亮的随盘的说明页包含了安装指南,该安装指南使您安装系统比直接用FTP安装更简单.

  • 我们自己的ports 树 该树自OpenBSD 3.3后,增加了许多. 几乎所有的软件包可以工作在几乎所有的硬件体系结构下.
  • 多个为大部分通用硬件体系结构预先构建的二进制软件包,可以非常方便的直接从光盘拷贝安装.
  • 和以前一样,还有不干胶贴图!

OpenBSD 3.3: May 1, 2003


To order a 3.3 CDROM, click here.
  • Three CDROMs in a jewel case.
  • The complete install components for FIVE architectures: i386, macppc, vax, sparc, sparc64.
  • The following architectures only available via FTP download: alpha, hppa, hp300, mvme68k, mac68k.
  • The CDs are bootable on i386, macppc, sparc, and sparc64.
  • A funky and surprisingly artistic CD insert sheet which contains installation instructions. The information on this piece of paper makes OpenBSD somewhat easier to install than if you do an FTP install.
  • A full source tree (ready for AnonCVS use).
  • The latest reliable XFree86 binaries for all architectures
  • The latest XFree86 source code with small modifications by us to make it prettier.
  • Our own ports tree which has improved an insane amount since OpenBSD 3.2. Almost all packages work on almost all architectures.
  • Several pre-built binary packages for the most common architectures, which are very easy to install directly off the CDROM.
  • As always, stickers included!
  • And many other things...
