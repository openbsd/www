OpenCVS sera disponible bientôt.

OpenCVS est une implémentation LIBRE de Concurrent Versions System, le plus célèbre logiciel libre de gestion de versions. Il peut être aussi bien utilisé en tant que client ou serveur pour des dépôts et fournit un contrôle d'accès granulaire pour les données stoquées dans le dépôt. Il cherche autant que possible à être compatible avec les autres implémentations de CVS, sauf lorsque certaines fonctionnalités réduisent la sécurité du système.

Le projet OpenCVS a été démarré suite aux discussions suivant les dernières vulnérabilités annoncées de GNU CVS. Bien que CVS soit largement utilisé, son développement a presque cessé ces dernières années et plusieurs problèmes de sécurité sont apparus, aussi bien dans l'implémentation que dans les mécanismes.

OpenCVS est développé par le Projet OpenBSD. Le logiciel est librement utilisable et réutilisable par tous, sous une licence BSD.
