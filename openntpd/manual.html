Manual pages

Web manual pages are available from OpenBSD. These manual pages reflect the latest development release of OpenNTPD.

  • ntpd - Network Time Protocol daemon
  • ntpd.conf - Network Time Protocol daemon configuration file
The NTP protocol implemented in OpenNTPD is described in RFC 1305 and RFC 2030. A reference implementation and other documentation of the NTP protocol is available from ntp.org.
