Зеркала

Исходный код OpenNTPD 4.6 был опубликован в день его официального релиза: 1 ноября 2009 года

Исходный код написанный специально для OpenBSD чист, минимален и прост для аудита. Отдельная группа программистов работает над портируемой версией OpenNTPD, которая доступна для других операционных систем.

Повторяем: для других операционных систем используйте портируемую версию!
