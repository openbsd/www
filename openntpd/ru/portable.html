Портируемая версия OpenNTPD

Проект OpenNTPD производит очень маленькую, безопасную и легкую в поддержке версию для проекта OpenBSD. Группа разработчиков, занимающаяся портированием OpenNTPD на другие операционные системы, берет эту упрощенную (базовую) версию и добавляет код, обеспечивающий работу OpenNTPD на многих других операционных системах.

Предоставляемая версия OpenNTPD работает на следующих Unix-подобных операционных системах:

  • FreeBSD
  • NetBSD
  • Linux
  • Solaris
  • Mac OS X
  • HP-UX

Портируемый OpenNTPD следует разработке официальной версии, но выпуски не синхронизируются. Портируемые выпуски помечаются литерой 'p' (например 3.6p1). Официальный исходный код никогда не будет использовать суффикс 'p'.
