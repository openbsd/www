すんごく長いあいだ、デフォルトインストールでのリモートセキュリティホールはふたつだけ!

OpenBSD プロジェクトでは、フリーでマルチプラットホーム対応の 4.4BSD ベースの UNIX 系 OS を提供しています。私たちのプロジェクトが特に力を入れているのは、 移植性、標準化、正当性、積極的なセキュリティ そして 暗号機能の統合です。

OpenBSD は私たちの FTP サイトから自由に入手できるほか、 お値ごろな 3 枚組 CD セットもあります。

最新のリリースは OpenBSD 5.2 です。 これは 2012年 11月1日にリリースされました。

OpenBSD はすべてボランティアによって開発されています。 本プロジェクトは開発環境と 開発者のイベント の 資金を受け持っており、これらは 多くの店舗で販売されている CD の売り上げや、 個人または団体からの 寄付 によってまかなわれています。 これらの金銭的援助のおかげで、OpenBSD が存在し続けて、みんなの好きなように使ったり再利用したりすることが フリーでありつづけることが保証されるのです。 また、 Tシャツ と ポスター も購入できます。

姉妹プロジェクト: OpenSSH, OpenBGPD, OpenNTPD, OpenCVS, OpenSMTPD
