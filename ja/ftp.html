構成・レイアウト:

各サーバのファイルの階層構造はおおまかに次のとおり:

  • .../OpenBSD/3.1/
    ここに最も最近のリリースが置かれている。
    • alpha/
    • amiga/
    • hp300/
    • i386/
    • mac68k/
    • macppc/
    • mvme68k/
    • sparc/
    • sparc64/
    • vax/
    • src.tar.gz
       /usr/src に置かれるユーザランドソースコード。 (これにはカーネルソースコードは含まれていない)
    • srcsys.tar.gz
       /usr/src/sys に置かれるカーネルソースコード。
    • ports.tar.gz
       /usr/ports に置かれる ports のソースコード。
    • XF4.tar.gz
       XFree86 (X11) ソースコード。

  • .../OpenBSD/snapshots/
    主なアーキテクチャ用にだいたい毎月ビルドされる、 安定性/品質を保証しないミニリリースの試み。 これは、我々はテストリリースと位置付けている。

  • .../OpenBSD/src/
    個々のファイルにアクセスしたい人のための、CVS でチェックアウトされた 毎日のソースツリー。通常は、CTM か、 ここに位置するミラーの代わりに AnonCVS の使用をお勧めする。つまり、すべての FTP ミラーが、この場所を ミラーしているわけではない、ということだ。

  • .../OpenBSD/tools/
    その他もろもろ。
  • .../OpenBSD/3.0/
    上述した 3.1 リリースと同様のレイアウト。懐古趣味用。
  • .../OpenBSD/2.9/
    上述した 3.1 リリースと同様のレイアウト。懐古趣味用。
  • .../OpenBSD/2.8/
    上述した 3.1 リリースと同様のレイアウト。懐古趣味用。
  • .../OpenBSD/2.7/
    上述した 3.1 リリースと同様のレイアウト。懐古趣味用。
  • .../OpenBSD-ISO/
    CD セットにパッケージされなかったリリースの特別な部分のための ISO イメージ。現時点では、3.1 リリースの 4 枚目の CD のみを含む。
    通常は ftp ミラーはこのディレクトリを持っていくことはないことに注意。

新しく FTP/ANONCVS ミラーを立てようと する人なら、次の ftp://ftp.openbsd.org:/pub/OpenBSD/ で使用される ディスク使用量の見積りに興味があるだろう。 


                      現時点    可能性
pub/OpenBSD/2.7        1.4GB     1.4GB   (OpenBSD 2.7 リリース)
pub/OpenBSD/2.8        2.0GB     2.0GB   (OpenBSD 2.8 リリース)
pub/OpenBSD/2.9        3.0GB     3.0GB   (OpenBSD 2.9 リリース)
pub/OpenBSD/3.0        4.5GB     4.5GB   (OpenBSD 3.0 リリース)
pub/OpenBSD/3.1        6.9GB     6.9GB   (OpenBSD 3.1 リリース)
pub/OpenBSD/snapshots  8.5GB さらに巨大! (現在のスナップショット、
                                          通常は最新リリース以上のサイズとなる)
pub/OpenBSD/src        0.6GB     0.7GB   (FTP 可能なソースツリー、ANONCVS から
                                          夜毎チェックアウトされているだけのもの
                                          なので、たぶん、これを得るための FTP
                                          ミラーの使用は避けるべきで、最寄りの
                                          ANONCVS サーバから単にアップデートする
                                          のが良いだろう。)
                     -----------------
pub/OpenBSD/          26.9GB    29.0GB   (全 FTP ミラーディスク使用量)
anoncvs                0.8GB     1.0GB   (ANONCVS ディスク使用量)
                     -----------------
                      27.7GB    30.0GB   (全ミラーディスク使用量)
また、pub/OpenBSD の下は、次のとおり。 
pub/OpenBSD/distfiles  4.4GB さらに巨大! (ports サブシステムの makefile が使用する
                                          配布ファイル。他の理由で必要でもない
                                          限りは、たぶん、これらのファイルを
                                          ミラーすべきではない。ports の
                                          makefile はこれらのためにデフォルトで
                                          ftp.openbsd.org をチェックする)

もし、新規に FTP, HTTP, AFS や RSYNC ミラーサイトを追加する場合には、 FTP 保守担当者に連絡して欲しい。
