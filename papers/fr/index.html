Document : Ouvrir l'Arborescence des Sources Avec Anonymous CVS

Ouvrir l'Arborescence des Sources Avec Anonymous CVS, Charles D. Cranor et Theo de Raadt
Anonymous CVS est un mécanisme avancé de distribution de fichiers source que nous avons créé afin de permettre aux projets de logiciels libres de distribuer le code source et des informations sur ce code aux utilisateurs Internet. Construit au dessus du système de contrôle des révision "Concurrent Versions System (CVS)", Anonymous CVS permet de manière sécurisée un accès anonyme en lecture seule à une arborescence de sources CVS. Ce document existe aussi au format PDF : anoncvs-paper.pdf Slides accompagnant le document sur précité. Présentés durant USENIX 1999 en Juin 1999 à Monterey en Californie. Ces slides existent aussi au format PDF.

Système De Gestion Des Mots De Passe A Long Terme, Niels Provos, David Mazieres
Plusieurs systèmes d'authentification dépendent de mots de passe secrets. Malheureusement, la longueur et l'aléa de mots de passe choisis par les utilisateurs restent inchangés dans la durée. Ce document discute de méthodes de construction de systèmes dans lesquels la sécurité des mots de passe peut faire face aux performances matérielles. Ce document existe aussi au format PDF : bcrypt-paper.pdf Slides accompagnant le document précité. Présentés lors de USENIX 1999 en juin 1999 à Monterey, Californie. Ces slides existent au format PDF : bcrypt-slides.pdf

Cryptographie dans OpenBSD : Une vue d'ensemble, Theo de Raadt, Niklas Hallqvist, Artur Grabowski, Angelos D. Keromytis, Niels Provos
Ce document donne une vue d'ensemble de la cryptographie utilisée dans OpenBSD. Il discute de composants variés tels que IPsec, les librairies SSL, le chiffrement fort de mots de passe, Kerberos IV, les générateurs de nombres aléatoires etc, et leur rôle dans la sécurité système y compris leurs interactions avec le reste des composants du système. Ce document existe aussi au format PDF : crypt-paper.pdf

Slides accompagnat le document précité. Présentés lors de USENIX 1999 en Juin 1999 à Monterey, Californie. Ces slides existent aussi au format PDF.
