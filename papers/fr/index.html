Présentation : AUUG 2002

Tutoriel : Protocole SSH et astuces, Damien Miller
Slides et notes de tutoriel écrites durant une session d'une demi-journée sur le protocole SSH et les choses sympathiques qu'on peut faire avec. Tous les exemples utilisent OpenBSD (bien évidemment). Cette présentation a été effectuée lors de la conférence de l'hiver 2002 du "Australian Unix Users Group". Les slides constituent une brève introduction au protocole, alors que les notes couvrent des exemples d'utilisation et des astuces. Voici une archive TAR compressée ainsi qu'un fichier postscript de la présentation pour une consultation hors ligne. Les notes existent en version Postscript ou PDF.

Présentation : USENIX 1999

Ouvrir l'Arborescence des Sources Avec Anonymous CVS, Charles D. Cranor & Theo de Raadt
Slides pour accompagner le document sur Anonymous CVS. Présentés lors de USENIX 1999 en juin 1999 à Monterey en Californie. Les mêmes slides existent au format PDF : anoncvs-slides.pdf

Document : Ouvrir l'Arborescence des Sources Avec Anonymous CVS

Ouvrir l'Arborescence des Sources Avec Anonymous CVS, Charles D. Cranor et Theo de Raadt
Anonymous CVS est un mécanisme avancé de distribution de fichiers source que nous avons créé afin de permettre aux projets de logiciels libres de distribuer le code source et des informations sur ce code aux utilisateurs Internet. Construit au dessus du système de contrôle des révision "Concurrent Versions System (CVS)", Anonymous CVS permet de manière sécurisée un accès anonyme en lecture seule à une arborescence de sources CVS. Ce document existe aussi au format PDF : anoncvs-paper.pdf

Présentation : USENIX 1999

Système De Gestion Des Mots De Passe A Long Terme, Niels Provos, David Mazieres
Slides accompagnant le document sur un système de gestion des mots de passe à long terme. Présentés lors de USENIX 1999 en juin 1999 à Monterey en Californie. Ces slides existent aussi au format PDF : bcrypt-slides.pdf

Document : Politique De Gestion Des Mots De Passe A Long Terme

Système De Gestion Des Mots De Passe A Long Terme, Niels Provos, David Mazieres
Plusieurs systèmes d'authentification dépendent de mots de passe secrets. Malheureusement, la longueur et l'aléa de mots de passe choisis par les utilisateurs restent inchangés dans la durée. Ce document discute de méthodes de construction de systèmes dans lesquels la sécurité des mots de passe peut faire face aux performances matérielles. Ce document existe aussi au format PDF : bcrypt-paper.pdf

Présentation : USENIX 1999

Cryptographie dans OpenBSD : Une vue d'ensemble, Theo de Raadt, Niklas Hallqvist, Artur Grabowski, Angelos D. Keromytis, Niels Provos
Slides accompagnant le document intitulé "cryptographie dans OpenBSD : une vue d'ensemble". Présentés lors de USENIX 1999 en juin 1999 à Monterey, Californie. Ces slides existent au format PDF : crypt-slides.pdf

Cryptographie dans OpenBSD : Une vue d'ensemble

Cryptographie dans OpenBSD : Une vue d'ensemble, Theo de Raadt, Niklas Hallqvist, Artur Grabowski, Angelos D. Keromytis, Niels Provos
Ce document donne une vue d'ensemble de la cryptographie utilisée dans OpenBSD. Il discute de composants variés tels que IPsec, les librairies SSL, le chiffrement fort de mots de passe, Kerberos IV, les générateurs de nombres aléatoires etc, et leur rôle dans la sécurité système y compris leurs interactions avec le reste des composants du système. Ce document existe aussi au format PDF : crypt-paper.pdf
