Productos para OpenBSD

OpenBSD anima a las compañías y a los desarrolladores independientes a que programen productos que puedan usarse con OpenBSD, o que estén basados en OpenBSD.

Existe un servicio de asistencia comercial de asistencia para OpenBSD disponible en nuestra lista de distribuidores de nuestra página de asesoría.

  • Sistemas OpenBSD preinstalados

    • Productos basados en OpenBSD

  • ePipe by Stallion Technologies
    Stallion Technologies usa OpenBSD como sistema operativo integrado en su pasarela de conexión de redes para medianas y pequeñas empresas. Esta pasarela tiene como fin el abaratamiento de los altos costes que supone la conexión de Redes Virtuales Privadas para las pymes, y para proveedores de servicios de aplicaciones que ofrezcan aplicaciones hospedadas a las pymes. ePipe funciona sobre un procesador Motorola ColdFire 5307 VL-RISC, y usa el software de cifrado integrado en OpenBSD para obtener conexiones seguras en ambas partes de la comunicación.

  • nCircle Network Security
    nCircle Network Security (antes Hiverworld) provee una amplia cobertura de seguridad en red, yendo más allá de la detección de intrusos para ofrecer un sistema de prevención de intrusiones inteligente. Su solución IP360, pendiente de patente, examina las vulnerabilidades de forma continua, sin invasiones, se combina con la monitorización completa del tráfico y el análisis de intrusiones. nCircle ha escogido el sistema operativo OpenBSD para su Sensor de Escaneos, Sensor de Intrusiones y Máquina de Consola que funcionan todos en una aplicación de red segura.

  • RTMX
    RTMX venden una versión de OpenBSD con complementos de funcionalidades POSIX a tiempo real añadidas. De forma desinteresada, nos han donado el código fuente para estas extensiones, y pronto serán integradas en OpenBSD.

  • syscall Network Solutions
    Syscall Network Solutions produce el dispositivo de seguridad syswall, basado en OpenBSD. Este aparato de seguridad contiene un cortafuegos con un filtro de paquetes con mantenimiento de estado, un sistema de detección de intrusos, y capacidad para Redes Privadas Virtuales (VPN) incluso con direcciones IP dinámicas. Ha sido diseñado especialmente para servicios de seguridad gestionados y servicios de seguridad monitorizados por nuestros expertos en seguridad informática. syswall también se puede usar como un enrutador de ADSL o cable, ya que contiene soporte para PPPoE y DHCP.

    • Productos de Hardware/Software para OpenBSD

  • PowerCrypt by GTGI
    Global Technologies Group, Inc. produce PowerCrypt, una placa PCI bus-mastering, que incorpora el procesador de cifrado Hifn 7751. El soporte es a nivel del núcleo del sistema y ha sido desarrollado por OpenBSD, con los controladores de dispositivos y el código de IPsec escrito por miembros de la comunidad de OpenBSD.

  • RedHat's Stronghold Secure Web Server
    RedHat dispone de un dæmon de HTTPS con criptografía de 128 bits, basado en el popular servidor de web Apache sobre OpenBSD, con Stronghold 3, su conocido servidor seguro para web.

  • Open Sound System de 4Front (sólo i386)
    4Front Technologies dispone de una versión comercial bastante modificada de los controladores de audio para OpenBSD/i386. Estos controladores substitutivos son compatibles con los controladores de audio que vienen con OpenBSD/i386, pero tienen soporte para muchas tarjetas adicionales y aprovechan al máximo las características de estas tarjetas. Este paquete se encuentra disponible para OpenBSD/i386 3.x.

  • Core Systems
    Core Systems vende InSite, una utilidad de estadísticas de web para servidores, fácil de usar, para OpenBSD/i386. InSite es parecido a otros productos como WebTrends, pero también se puede configurar a través de una interfaz de web para generar informes al vuelo, usando muy poco tiempo de CPU (Core puede proveer InSite para otras plataformas que no sean i386 por encargo).

  • Software2Go Motif (sólo i386 y SPARC)
    Software2Go, LLC ofrece "Motif 2.1.20 Development and Runtime", unos juegos de herramientas de tiempo real y desarrollo en Motif para OpenBSD.
    La distribución en desarrollo es para desarrolladores de software que hagan aplicaciones de interfaces gráficas de usuario con el estilo de Motif. La distribución de tiempo real se necesita para distribuir aplicaciones solo de binarios.
    Contacto: Software2Go, LLC Online Ordering, 817-431-8775 (phone/fax)

  • Zeus Web Server (sólo i386)
    Zeus Technology produce Zeus Web Server, el software de servidores de web de alto rendimiento con la más alta escalabilidad del mundo, para OpenBSD/i386 2.8 y 2.9. Unos cuantos ejemplos de sus excepcionales funcionalidades son: alta escalabilidad, estadísticas basadas en tiempo real en web, aceleración de ancho de banda por servidor, CGI seguros, soporte para agrupamientos (clustering), y FastCGI.
