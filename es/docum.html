Árbol de GNU Info

  • Info - Sistema de navegación de documentos.
  • Bison - Analizador de C compatible con yacc.
  • CPP - Preprocesador de C de GNU.
  • Emacs - Editor de textos extensible auto-documentado.
  • Forms mode - Modo de edición basado en formularios.
  • G++ & GCC - Compiladores de C++ y C de GNU.
  • G77 - Compilador de Fortran de GNU.
  • GDB - Depurador de código C.
  • Gawk - Implementación de AWK de GNU.
  • Gnus - lector/cliente de correo y noticias.
  • Gprof - Generador de perfiles de C/Pascal/Fortran77 de GNU.
  • libstdc++ - Biblioteca estándar de C++.
  • Make - Implementación de Make de GNU.
  • rx & regex - Bibliotecas de expresiones regulares.
  • Tar - Archivador Tar de GNU.
  • Termcap - Biblioteca que permite que las aplicaciones puedan manejar todo tipo de terminales de caracteres.
  • Texinfo - Obtención de un manual para impresión (con TeX) o de un fichero Info (con texinfo) desde un mismo archivo fuente.
  • VIP - Modo de emulación de Vi para Emacs.
  • Viper - Modo de emulación de Vi para Emacs más actual.
  • VM - Interfaz de correo (buzones compatibles con mailx y elm).
  • Calc - Calculadora para Emacs de GNU -- una calculadora de numérica y simbólica basada en RPN.
  • Elisp - Cómo extender Emacs.
  • Babyl - Formato (BABYL) de ~/RMAIL, usado por rmail de emacs.
  • Perl - Documentación de Perl.
  • GNATS - Sistema para informes de errores de GNU.
  • groff - Implementación de troff de GNU.
  • screen - Gestor de terminales virtuales.
