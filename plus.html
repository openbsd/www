OpenBSD -current Changelog

This selection is intended to include all important and all user-visible changes. For a complete record of all changes, please see the "source-changes" mailing list, called "OpenBSD CVS" in the archives, or use CVS.

Changes made between OpenBSD 7.9 and -current

  • Enable 64 bit DMA transfers on ix(4), ixv(4), ixl(4), igc(4), ice(4) and iwx(4).
  • Make httpd error out on presence of Content-Length and Transfer-Encoding headers for GET, HEAD and other methods that should have no body.
  • Enable forced resolvers that aren't in a preference block in unwind.
  • Enable 64-bit DMA transfers on the PCIe variants of re(4) cards.
  • Retire pledge(2) 'tmppath'.
  • Raise rad(8) lifetimes for the router, dns and nat64 to 60 minutes and lower the prefix valid lifetime to 60 minutes.

  • Add sorting (-O flag) and a custom format (-F) to tmux(1) list-keys.

  • Report success in ftp(1) when file is fully retrieved
  • Make bse(4) interrupts mp-safe.
  • Prevent false ELOOP error in socket splicing.
  • Print both nat-to and rdr-to in pfctl(8) show rules.
  • Prevent vmd(8) pause deadlock when vcpu doesn't halt.
  • Add relayd(8) support for PROXY protocol in TCP relays.
  • Make IPQoS first-match-wins in sshd_config(5).
  • Make sure that internal counters do not go out of bounds if the '-n' or '-A' traceroute(8) options are specified more than once.
  • Introduce swpskip to count how many times pages are not sent to swap because the pagedaemon detects the swap system won't be able to deliver results.
